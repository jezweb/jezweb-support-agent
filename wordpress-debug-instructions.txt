WordPress Debug Mode Setup Instructions
==========================================

1. Open your wp-config.php file (in the root of your WordPress installation)

2. Find the line that says: /* That's all, stop editing! Happy publishing. */

3. ADD these lines BEFORE that line:

// Enable WP_DEBUG mode
define( 'WP_DEBUG', true );

// Enable Debug logging to /wp-content/debug.log
define( 'WP_DEBUG_LOG', true );

// Hide errors from displaying on the site
define( 'WP_DEBUG_DISPLAY', false );
@ini_set( 'display_errors', 0 );

4. Save the file

5. Try activating the Jezweb Support Agent plugin again

6. Check for errors in: /wp-content/debug.log

7. Send me the error messages from debug.log so I can fix the issue!

Note: After debugging, you can disable debug mode by setting WP_DEBUG to false or removing these lines.
